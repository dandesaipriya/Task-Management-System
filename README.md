# Task Management System

A simple Task Management System built using React and Material UI.
The application allows users to add and delete tasks dynamically with a clean and responsive user interface.

## 🚀 Features

* Add new tasks
* Delete existing tasks
* Responsive UI using Material UI
* Styled with CSS and linear gradient background
* State management using React Hooks

## 🛠️ Technologies Used

* React (Vite)
* JavaScript
* Material UI
* CSS

## 📂 Project Structure

src/
App.jsx
App.css
main.jsx

## ▶️ Run Locally

Install dependencies

npm install

Start development server

npm run dev

Open in browser

http://localhost:5173

## 🌐 Deployment

The project is deployed using GitHub and Vercel.
